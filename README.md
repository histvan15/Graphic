# Az Elveszett Sziget

## A projekt leírása
Ez a program egy interaktív, 3D-s virtuális tér, amely egy elhagyatott szigetet mutat be. A felhasználó belső nézetből szabadon bejárhatja a területet, miközben a napszak dinamikusan változtatható.

  **Objektumok:**
  
  **Animációk:** tenger hullámzása
  
  **Textúrák**
  
  **Fények:** Lesz egy Nap aminek a fényét + és - állíthatjuk
  
  **Útmutató:** F1 gombbal megjelennek a mozgás,fényerősség és kilépés útmutatói
